# Room Functions Guide

- **Laboratory** - 50 sq ft per student

### 💼 Office/Administrative

### 🏥 Healthcare/Medical

### 🔧 Storage/Utility

## 🔍 Can't See All Options?

If you're not seeing all 20 options in the dropdown:

1. **Check if the dropdown is scrollable** - With 20 items, you may need to scroll within the dropdown
2. **Refresh the page** - Sometimes the browser cache needs to be cleared
3. **Try keyboard navigation** - Click the dropdown and use arrow keys to navigate through all options

## 📐 Automatic Occupancy Calculation

Each room type automatically calculates occupancy based on California Building Code (CBC) standards:
- The system divides the square footage by the occupancy factor
- You can manually override the calculated occupancy if needed
- The calculation appears with a calculator icon (🧮) next to the field

## 💡 Examples

- A **500 sq ft Classroom** = 25 students (500 ÷ 20)
- A **200 sq ft Restroom** = 5 people (200 ÷ 40)  
- A **1000 sq ft Storage** = 3 people (1000 ÷ 300)
- A **300 sq ft Other** = 3 people (300 ÷ 100)

The "Other" option is perfect for any room type not specifically listed!
